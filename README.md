# 系统监控与进程管理工具 v2.0

一个使用 PySide6 开发的现代化系统监控和进程管理工具，提供专业的图形化界面来实时监控系统资源、管理进程以及查看详细的系统信息。

# 进程管理工具

1、可以查看和管理进程
2、可以查看和管理端口
3、可以查看和管理服务
4、可以查看计算机系统信息
5、可以查看资源利用率、

跨平台的服务、端口、进程管理工具，可以查看系统信息。

支持Windows、Linux、MacOS

支持Hostfile的管理

支持网络管理，查看网络信息，连接WIFI等

## 功能特性

### 🖥️ 系统概览
- **实时监控**: CPU使用率、内存使用率、磁盘使用率
- **基本信息**: 计算机名称、操作系统、系统架构、处理器信息
- **可视化显示**: 使用进度条和图表直观显示系统状态
- **系统启动时间**: 显示系统启动时间和运行时间
- **实时统计**: 实时进程数统计

### 📊 进程管理
- **进程列表**: 显示所有运行中的进程
- **详细信息**: PID、进程名、CPU使用率、内存使用率、磁盘读取量、运行状态
- **进程控制**: 可以结束选中的进程（带确认对话框）
- **搜索过滤**: 支持进程搜索和过滤功能
- **详细视图**: 显示详细的进程信息（创建时间、线程数、父进程等）
- **排序功能**: 可排序的表格视图，支持按各列排序
- **实时刷新**: 支持手动刷新和自动刷新进程列表

### 🌐 网络端口监控
- **网络连接**: 显示所有活动的网络连接
- **连接详情**: 协议类型、本地地址、远程地址、连接状态、关联PID
- **端口监控**: 监控系统开放的端口

### 🔧 硬件信息
- **CPU信息**: 核心数、频率信息
- **内存信息**: 总内存、可用内存、使用情况、交换内存
- **磁盘信息**: 所有磁盘分区的使用情况
- **网络接口**: 网络适配器信息、IP地址、MAC地址

### 📋 系统详情
- **系统信息**: 详细的操作系统信息
- **用户信息**: 当前登录用户
- **启动时间**: 系统启动时间
- **环境变量**: 重要的系统环境变量

### 🎨 用户界面
- **现代化设计**: 使用PySide6创建美观的用户界面
- **选项卡布局**: 分离不同功能模块，便于使用
- **菜单栏和工具栏**: 提供快捷操作
- **状态栏**: 显示实时状态信息
- **自动刷新**: 可开关的自动刷新功能
- **响应式设计**: 适应不同屏幕尺寸

## 安装和运行

### 环境要求
- Python 3.8 或更高版本
- Windows、macOS 或 Linux 操作系统

### 依赖包
- **PySide6 >= 6.5.0**: Qt6 的 Python 绑定，用于创建现代化的GUI界面
- **psutil >= 5.9.0**: 跨平台的系统和进程监控库

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序

#### 方法1：使用Python直接运行
```bash
python main.py
```

#### 方法2：使用系统信息工具（如果存在）
```bash
python system_info_tool.py
```

#### 方法3：使用批处理文件（Windows）
```bash
run.bat
```

## 使用说明

### 进程管理选项卡
1. 启动程序后默认显示"进程管理"选项卡，展示当前系统的所有进程
2. 在搜索框中输入进程名称可以实时过滤显示特定进程
3. 点击表格标题可以按该列排序（PID、CPU使用率、内存使用率等）
4. 选择一个进程后，可以：
   - 点击"结束进程"按钮终止该进程（需要确认）
   - 点击"进程详情"按钮查看详细信息
   - 使用工具栏或菜单栏的快捷操作
5. 程序每3秒自动刷新进程列表，也可以手动刷新

### 系统信息选项卡
1. 切换到"系统信息"选项卡查看系统资源使用情况
2. 左侧显示系统概览信息（操作系统、处理器、启动时间等）
3. 右侧显示实时资源使用率（CPU、内存、磁盘）
4. 底部显示详细的系统统计信息
5. 系统信息每2秒自动更新

### 网络端口监控
- 显示TCP和UDP连接
- 监控端口占用情况
- 显示连接状态和关联进程

### 硬件信息查看
- 查看CPU详细信息和性能数据
- 监控内存使用情况和交换内存
- 查看所有磁盘分区的使用情况
- 显示网络适配器信息

### 快捷键
- F5：刷新进程列表
- Ctrl+Q：退出程序

### 菜单功能
- 文件菜单：刷新、退出
- 视图菜单：开关自动刷新
- 帮助菜单：关于程序

## 注意事项

1. **权限要求**: 某些功能可能需要管理员权限才能正常工作
2. **性能影响**: 程序会定期获取系统信息，对系统性能影响很小
3. **兼容性**: 在不同操作系统上某些功能可能有差异
4. **进程安全**: 终止系统关键进程可能会导致系统不稳定，请谨慎操作
5. **权限限制**: 某些进程可能需要管理员权限才能终止
6. **建议**: 建议以管理员身份运行程序以获得完整功能
7. **磁盘监控**: 在Windows系统上，磁盘使用率显示的是根目录（C:）的使用情况
8. **多线程**: 程序使用多线程技术，系统信息更新不会影响界面响应性

## 故障排除

### 常见问题

**Q: 程序无法启动**
A: 请确保已安装所有依赖包，并使用Python 3.8或更高版本

**Q: 某些进程信息显示为N/A**
A: 这通常是由于权限不足导致的，可以尝试以管理员身份运行程序

**Q: 网络信息不完整**
A: 某些网络连接信息可能需要更高权限才能访问
